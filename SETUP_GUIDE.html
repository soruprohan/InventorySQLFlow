‚úÖ Prerequisites

Make sure you have the following installed:

  • XAMPP (includes Apache & MySQL)
  • Modern Web Browser (Chrome, Firefox, Edge)
  • Text Editor (optional, for customization)

üìã Installation Steps

Step 1: Start XAMPP Services

  1. Open XAMPP Control Panel
  2. Click Start for Apache
  3. Click Start for MySQL
  4. Both should show green "Running" status

Step 2: Create Database

Option A: Using phpMyAdmin (Recommended)

  1. Open browser and go to: http://localhost/phpmyadmin
  2. Click on "New" in the left sidebar
  3. Database name: inventory_system
  4. Click "Create"
  5. Select the inventory_system database
  6. Click on "SQL" tab
  7. Open install_database.sql file from this project
  8. Copy all content and paste it in the SQL tab
  9. Click "Go" to execute
  10. You should see "Database schema created successfully!"

Option B: Using Command Line

cd C:\xampp\mysql\bin
mysql -u root -p
CREATE DATABASE inventory_system;
USE inventory_system;
SOURCE C:/xampp/htdocs/inventoryflowv2/install_database.sql;
EXIT;

Step 3: Verify Database Configuration

Open api/config.php and verify the settings:

define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');  // Leave empty if no password set
define('DB_NAME', 'inventory_system');

If your MySQL has a password, update DB_PASS accordingly.

Step 4: Access the Application

  1. Open your web browser
  2. Navigate to: http://localhost/inventoryflowv2/
  3. You should see the Inventory Management System dashboard
  4. The SQL Monitor dialog should appear in the bottom-right corner

‚úÖ Verification

To verify everything is working:

  1. Dashboard should show statistics (products, inventory, suppliers, orders)
  2. Click on different sections (Products, Suppliers, etc.)
  3. Watch the SQL Monitor - it should display queries as you navigate
  4. Try adding a new supplier or product
  5. Check the SQL Query History in the monitor

üéì Using the System

Understanding the SQL Monitor

  • Current Query: Shows the most recent SQL executed
  • History Button: View all SQL queries from this session
  • Copy Button: Copy query to clipboard
  • Execute Button: Run the query in SQL Executor
  • Clear Button: Clear the history
  • Minimize Button: Hide/show the monitor

SQL Executor Tab

Navigate to the "SQL Executor" tab to:

  • Write custom SQL queries
  • Test SELECT, INSERT, UPDATE, DELETE operations
  • See real-time results
  • Learn by experimentation

Example queries to try:

-- View all products with their categories
SELECT p.product_name, c.category_name, p.unit_price 
FROM products p 
LEFT JOIN categories c ON p.category_id = c.category_id;

-- Count products by category
SELECT c.category_name, COUNT(p.product_id) as count 
FROM categories c 
LEFT JOIN products p ON c.category_id = p.category_id 
GROUP BY c.category_id, c.category_name;

-- View low stock products
SELECT * FROM vw_low_stock_products;

-- Products above average price
SELECT product_name, unit_price 
FROM products 
WHERE unit_price > (SELECT AVG(unit_price) FROM products);

  7. Advanced Queries: Try aggregate functions, subqueries, and views
