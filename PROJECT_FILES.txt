**api/transactions.php**
- GET: List transactions

- DELETE: Delete transaction

**api/adjustments.php**
- GET: List adjustments

- Demonstrates: Multiple related queries, HAVING

**api/execute_sql.php**
- POST: Execute raw SQL queries
- Query type detection
- Result formatting
- Security checks

## Quick Start Guide

### 1. Installation (5 minutes)
1. Open XAMPP Control Panel
2. Start Apache and MySQL
3. Open phpMyAdmin (http://localhost/phpmyadmin)
4. Create database: `inventory_system`
5. Import: `install_database.sql`

### 2. Configuration (1 minute)
1. Open `api/config.php`
2. Verify database credentials
3. Save if changes made

### 3. Launch (1 second)
1. Open browser
2. Navigate to: http://localhost/inventoryflowv2/
3. Start exploring!

### 4. Learning Path (Recommended Order)
1. Open **SETUP_GUIDE.html** for detailed instructions
2. Read **README.md** for features overview
3. Explore the **Dashboard** - see statistics queries
4. Navigate to **Products** - add/view products, watch SQL
5. Navigate to **SQL Executor** - try queries from **TEST_QUERIES.sql**
6. Check **SQL Monitor** - view history of all queries
7. Reference **SQL_REFERENCE.sql** - complete SQL guide
